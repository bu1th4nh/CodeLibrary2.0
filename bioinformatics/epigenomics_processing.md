# Epigenomic Data Preprocessing pipeline from fragment file to expression matrix


## Prequisites
- Python 3.11
- MACS3 library: `pip install MACS3`
- EpiScanpy library: `pip install episcapy`
- The following files are required for the pipeline:
    - Fragment files with suffix `fragments.tsv.gz`: the fragment file generated by Cell Ranger ATAC or similar pipelines, containing the genomic coordinates of the sequenced fragments.
    - Tabix index files with suffix `fragments.tsv.gz.tbi`: the index file for the fragment file, allowing for efficient querying of the fragments by genomic coordinates. 

## Step 1: Call peaks using MACS3
```python3
    cmd = [
        "macs3", "callpeak",
        "-t", "/path/to/fragments.tsv.gz",
        "-f", "BEDPE",                      # 10x fragments are fragment-level BEDPE-like
        "-g", "mm",                         # mouse genome size shortcut (GRCm38/mm10-ish)
        "-n", "<file_name_prefix>",         # prefix for output files
        "--outdir", "/path/to/output_dir",  # where to save the peak files
        "-q", "0.01",                       # whatever FDR you feel like; this is fine for TF-IDF+LSI
        "--buffer-size", "10000000"
    ]
    compl_process = subprocess.run(cmd)
```
This snippet will generate the following files in the output directory:
- `<file_name_prefix>_peaks.narrowPeak`: the called peaks in narrowPeak format
- `<file_name_prefix>_summits.bed`: the summit positions of the peaks
- `<file_name_prefix>_peaks.xls`: a detailed report of the peaks called



## Step 2: Generate peak-by-cell count matrix using EpiScanpy

### For ATAC-seq data
```python3
import episcanpy as epi
peak_data = epi.ct.load_features("/path/to/<file_name_prefix>_summit.bed")
adata = epi.ct.bld_mtx_fly(
    tsv_file="/path/to/fragments.tsv.gz",   # The tabix is also required in the same dir
    annotation=peak_data,
    genome=None,                            # or "GRCm38"/whatever you used
    save=False
)
```


### For Histone modification data
Instead of calling peaks, we will create fixed-size genomic bins to quantify the signal. Several bin sizes can be used, such as 1kb, 5kb, or 10kb, depending on the resolution required for the analysis.
```python3
import episcanpy as epi
peak_data = epi.ct.make_windows(size=5000, chromosomes="mouse")
adata = epi.ct.bld_mtx_fly(
    tsv_file="/path/to/fragments.tsv.gz",   # The tabix is also required in the same dir
    annotation=peak_data,
    genome=None,   # or "GRCm38"/whatever you used
    save=False
)
```
