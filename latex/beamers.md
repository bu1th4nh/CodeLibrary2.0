# Beamer slides 


## Footnotes

To insert a footnote mark into the text:
```latex
\footnotemark[1]
```

To insert the footnote text at the bottom of the page:
```latex
\footnotetext[1]{This is the footnote text.}
```



## Columns

```latex
\begin{columns}
    \begin{column}{0.5\textwidth}
        % First column content goes here
    \end{column}
    \begin{column}{0.5\textwidth}
        % Second column content goes here
    \end{column}
\end{columns}
```
